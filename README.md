# angular-exports-loader-example

Example webpack configuration to load CommonJS-incompatible AngularJS v1.3.0 via [exports-loader](https://github.com/webpack/exports-loader).

### Installation

```shell
npm install
```

### Usage

```shell
npm start
open http://localhost:8080
```
